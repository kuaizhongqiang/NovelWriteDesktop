# 页面布局

## 通用布局

```
┌─────────────────────────────────────────────────────┐
│  顶部导航栏                                           │
│  [Dashboard] [阅读] [设定] [角色] [大纲] [写作] [风格]  │
├──────────────────────────────┬────────────────────────┤
│                              │                        │
│         主内容区              │     Agent Bar           │
│                              │    （AI 助手面板）       │
│                              │    可收起               │
│                              │    左侧或右侧均可        │
│                              │                        │
│      主内容区功能区内可        │                        │
│      内部二分/三分布局         │                        │
│                              │                        │
└──────────────────────────────┴────────────────────────┘
```

## 导航

- **顶部导航栏**：固定在顶部，展示所有页面入口
- 当前页面高亮显示
- 点击跳转到对应页面

## Agent Bar

- 所有页面共用
- 位置：左侧或右侧均可，需可配置
- 可收起变窄（瘦）
- 后续 AI 功能入口

## 页面切换流程

```
Dashboard
  │
  ├─ 点击卡片 [阅读] → /novel/:id/read
  │
  └─ 点击卡片 [编辑] → /novel/:id/settings
                              │
                    ┌─────────┼─────────┐
                    ▼         ▼         ▼
                角色设定    大纲设定   写作编辑
                              │
                              ▼
                          风格设定
```
