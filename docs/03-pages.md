## 1. Dashboard（小说看板）

```
┌──────────────────────────────────────────┬────────────────┐
│  [Dashboard] [阅读] [设定] [角色] [大纲] │   Agent Bar    │
├──────────────────────────────────────────┤                │
│                                          │                │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐    │                │
│  │ 小说标题  │ │ 小说标题  │ │ 小说标题  │    │                │
│  │ 奇幻 · 剑 │ │ 都市 · 悬 │ │ 科幻 · 末 │    │                │
│  │ #穿越 #热 │ │ #推理   # │ │ #AI    # │    │                │
│  │ 2.3w 字   │ │ 8.7w 字   │ │ 1.1w 字   │    │                │
│  │ 🌍 2026/06│ │ 🔒 2026/05│ │ 🌍 2026/04│    │                │
│  │ [阅读]    │ │ [阅读]    │ │ [阅读]    │    │                │
│  │ [编辑]    │ │ [编辑]    │ │ [编辑]    │    │                │
│  └─────────┘ └─────────┘ └─────────┘    │                │
│                                          │                │
│  ┌──────────────┐                       │                │
│  │     +        │                       │                │

│  └──────────────┘                       │                │
│                                          │                │
```

**交互：**
- 卡片网格布局，自适应列数
- 点击 [阅读] → 阅读页面
- 点击 [编辑] → 基础设定页面
- 点击空白卡片 → 新建小说，跳转到基础设定

## 2. 阅读页面

```
┌──────────────────────────────────────────┬────────────────┐
│  [...导航...]                             │   Agent Bar    │
├──────────────────────────────────────────┤                │
│  [Aa ▼] 字体大小  [≡ 目录]  [上一章]  [下一章]              │
│                                          │                │
│          第三章 · 初见                    │                │
│                                          │                │
│   正文内容...                             │                │
│   正文内容...                             │                │
│   正文内容...                             │                │
│   正文内容...                             │                │
│                                          │                │
│       —— 本章完 · 共 2,341 字 ——          │                │
│                                          │                │
│  [← 上一章]                    [下一章 →]  │                │
│                                          │                │
└──────────────────────────────────────────┴────────────────┘
```

**交互：**
- 滚动阅读
- 字体大小可调节（预设几个档位或自由缩放）
- 快捷键：← 上一章 / → 下一章
- 目录浮窗：

```
┌─────────────────┐
│  📖 目录        │
│                 │
│  第一卷 少年出山  │
│  ├ 第一章 觉醒   │
│  ├ 第二章 初战   │
│  └▶ 第三章 初见  │
│  第二卷 风云起   │
│  ├ 第四章 入世   │
│  └ 第五章 暗流   │
│                 │
│     [✕ 关闭]    │
└─────────────────┘
```

- 目录浮窗：点击 ≡ 打开，点击章节跳转，点击外部或 ✕ 关闭
- 每章末尾显示字数

```
┌──────────────────────────────────────────┬────────────────┐
│  [...导航...]                             │   Agent Bar    │
├──────────────────────────────────────────┤                │
│  角色设定                                  │                │
│                                          │                │
│  ─── 男主角 ───                         │                │
│  姓名  [________________]                │                │
│  描述  [________________________________] │                │

│                                          │                │

│                         [+ 新增女主角]     │                │
│                                          │                │
│  ─── 配角 ───                           │                │
│  ┌──────────────────────────────────┐    │                │
│  │ [×] 姓名: 张三  描述: ... 关系: ... │    │                │
```

## 5. 大纲设定页面

```
┌──────────────────────────────────────────┬────────────────┐
│  [...导航...]                             │   Agent Bar    │
├──────────────────────────────────────────┤                │
│  大纲设定                                  │                │
│                                          │                │
│  主角金手指  [________________________]   │                │
│  世界观    [________________________]     │                │
│  写作要点  [________________________]     │                │
│                                          │                │
│  ─── 大纲结构 ───                       │                │
│                                          │                │
│  ┌─ ▼ 第一卷：少年出山                    │                │
│  │   描述：xxx...                        │                │
│  │                                       │                │
│  │  ├── 第一章 觉醒 [hook:...]           │                │
│  │  ├── 第二章 初战 [hook:...]           │                │
│  │  └── 第三章 初见 [hook:...]           │                │
│  │                  [+ 添加章节到此幕]    │                │
│  │                                       │                │
│  └─ [+] 添加新幕                          │                │
│                                          │                │
│  ┌─ ▼ 第二卷：风云起                      │                │
│  │   描述：xxx...                        │                │
│  │  ├── 第四章 入世 [hook:...]           │                │
│  │  └── [+] 添加章节到此幕                │                │
│  └─                                      │                │
│                                          │                │
│                    [✕ 取消]    [保存]     │                │
└──────────────────────────────────────────┴────────────────┘
```

**交互：**
- 手风琴结构：幕（Phase）可展开/折叠
- 每个幕内列出章节大纲
- 可新增/编辑/删除幕和章节大纲
- 每个章节大纲：标题、描述、hook

## 6. 写作编辑页面

```
┌──────────────────────────────────────────┬────────────────┐
│  [...导航...]                             │   Agent Bar    │
├───────────────────┬──────────────────────┤                │
│   大纲树           │    编辑器             │                │
│                   │                      │                │
│  📖 小说标题      │  Aa ▼  1,234 字      │                │
│                   │                      │                │
│  ┌─ 第一卷        │  第三章 · 初见        │                │
│  │  少年出山       │                      │                │
│  │ ├ 第一章       │  ┌──────────────────┐│                │
│  │ ├ 第二章       │  │                  ││                │
│  │ └▶ 第三章 ◀   │  │  正文写作区域     ││                │
│  └─ 第二卷        │  │  纯文本 textarea  ││                │
│    ├ 第四章       │  │                  ││                │
│    └ 第五章       │  │                  ││                │
│                   │  └──────────────────┘│                │
│                   │                      │                │
│                   │  [← 上一章] [保存] [下一章 →]         │
└───────────────────┴──────────────────────┴────────────────┘
```

**交互：**
- 左侧大纲树：展示幕→章节的层级结构
- 当前正在编辑的章节高亮
- 点击大纲树其他章节 → 切换到该章节编辑
- 右侧纯文本编辑器
- 顶部显示当前章节字数
- 手动保存按钮
