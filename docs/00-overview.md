# NovelWrite Desktop — 产品设计文档

## 项目说明

小说写作平台，支持多小说管理、角色设定、大纲规划、纯文本写作。
后续会引入 AI/LLM 辅助写作功能。

## 技术栈

- **前端：** Vue 3 + TypeScript + Vite 8 + Naive UI
- **服务端：** 待定（后续开发）

## 约定

- **纯文本写作：** 正文内容严格纯文本，不含 HTML 或 Markdown 等任何格式标记。
- **页面通用布局：** 主内容区（左/中） + Agent Bar（右，可收起）。
- **顶部导航栏：** 用于页面切换。

## 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | Dashboard | 小说卡片看板 |
| `/novel/:id/read` | 阅读 | 只读模式浏览小说 |
| `/novel/:id/settings` | 基础设定 | 小说标题/题材/标签/简介 |
| `/novel/:id/roles` | 角色设定 | 主角/女主角/配角管理 |
| `/novel/:id/outline` | 大纲设定 | 幕(Phase)→章节大纲管理 |
| `/novel/:id/write` | 写作编辑 | 大纲树 + 纯文本编辑器 |
| `/novel/:id/style` | 写作风格 | 字数/基调设置 |
