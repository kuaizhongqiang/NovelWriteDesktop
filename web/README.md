# NovelWrite Web

小说写作平台前端。

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 目录结构

```
src/
├── api/          # HTTP 请求封装
├── assets/       # 静态资源
├── composables/  # Vue 组合式函数
├── layouts/      # 布局组件
├── pages/        # 页面组件
├── router/       # 路由配置
├── stores/       # 状态管理
├── types/        # TypeScript 类型定义
├── utils/        # 工具函数
├── App.vue       # 根组件
└── main.ts       # 入口文件
```

## 技术栈

- Vue 3 + TypeScript
- Vite 8
- Naive UI
